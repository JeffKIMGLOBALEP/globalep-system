## 已验证的核心功能流程

1. **登录 → 主界面**: doLogin() → buildSidebar() → showPage('dashboard') ✅
2. **导航切换**: 点击侧边栏 → showPage() → renderPage() → 页面渲染+动画 ✅
3. **语言切换**: switchLang() → Store.lang更新 → 侧边栏重建 → 页面重载 ✅
4. **撤销/恢复**: undoAction/redoAction 基于 undoHistory/redoHistory 栈 ✅
5. **Excel导入**: importExcel() → FileReader → XLSX.read() → 数据解析 ✅
6. **Excel导出**: exportExcel() → 数据映射 → XLSX.writeFile() → 下载 ✅
7. **3D地球**: Three.js 初始化 → animateGlobe() 持续渲染 ✅
8. **通知系统**: toggleNotifications() → 下拉列表渲染 ✅
9. **模态框**: openModal()/closeModal() → 表单渲染 → 保存 ✅
10. **Toast提示**: toast() → 自动消失 ✅
